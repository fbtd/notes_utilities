# notes utilities

sort text from stdin, writes result to stout

## format
evernote (new web interface)
*tabs* are used as separator

```
section A
- point one
- point two

section B
- point one
	- point one - one
	- point one - two
		- point one - two - one
```

## options
* -s --sections: move sections around
